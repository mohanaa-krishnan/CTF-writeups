# Hash Cracker (Basic)

This is a simple Python-based hash cracking tool that supports:
- MD5
- SHA1
- SHA256

## Features
- Automatic hash detection
- Dictionary attack
- Simple and beginner-friendly


## Notes
- This tool uses dictionary attack only
- Works only for unsalted hashes
